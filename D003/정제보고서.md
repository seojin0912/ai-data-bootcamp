# 옷장패션 데이터 정제 보고서

## 1. 데이터 개요
- 행/열: 1,500 × 7
- 주요 컬럼: order_id, customer_age, category, channel, price, quantity, amount, return_amount

## 2. 진단 결과
- 결측: amount (3.40%, app 채널에 쏠림 → MAR 의심 가능), price(0.33%, 소수 — MCAR 가능)
- 이상치(IQR): customer_age(999·0, 18건), quantity(200, 1건), amount(50,000,000, 145건)
- 의심되는 결측 유형: amount는 app 채널에 쏠림 → MAR

## 3. 처리 결정과 근거
### amount 결측	
결정 채널별 중앙값 대체	
근거 app 결측 비율↑ → MAR 가설에 부합	
한계 채널 외 다른 원인은 검토 안 함

### price 결측	
결정 카테고리별 중앙값 대체
근거 비율 0.3%로 적음, 카테고리별 가격대 보존	
한계 표본 적어 카테고리 통계 불안정

### customer_age = 999, 0	
결정 NaN 표시 → 중앙값 대체	
근거 물리적 불가능 — 입력 실수 추정	
한계 추정은 추정일 뿐 외부 인증 데이터 없음

### quantity = 200	
결정 NaN 표시 → 중앙값 대체	
근거 일반 소비자 도메인 외 값	
한계 만약 도매 가능 고객이면 제거가 손실

### amount = 50,000,000	
결정 유지 + amount_outlier 플래그	
근거 1건의 비즈니스적 가능성 보존	
한계 도매면 별도 분석 필요

## 4. 처리 후 검증
